## 1.1 数字逻辑基础

### 模拟信号与数字信号

| 模拟信号 (Analog) | 数字信号 (Digital) |
|-------------------|-------------------|
| 连续变化的物理量 | 离散的量化值 |
| 理论上无限精度 | 有限精度（受字长限制） |
| 易受噪声干扰 | 抗干扰能力强 |
| 处理电路复杂 | 电路结构规整 |

#### 模拟信号
- 实际物理量多为模拟量（温度、压力、声音、图像等）
- 信号值在时间和幅度上连续

#### 数字信号
- 采用 "0" 和 "1" 两个逻辑电平表示信息
- 信号值在时间和幅度上离散
- 便于存储、传输和处理

### 逻辑电平

数字电路中用不同的电压范围表示逻辑 `0` 和逻辑 `1`：

| 逻辑值 | 正逻辑 | 负逻辑 |
|--------|--------|--------|
| **逻辑 1** | 高电平 (H) | 低电平 (L) |
| **逻辑 0** | 低电平 (L) | 高电平 (H) |

> 本课程默认采用**正逻辑**。

### 数字电路发展历史

- **1847 年**: George Boole 创立布尔代数
- **1938 年**: Claude Shannon 发表《继电器与开关电路的符号分析》，将布尔代数应用于开关电路
- **1946 年**: 第一台通用电子数字计算机 ENIAC 诞生
  - 使用 18,000 个真空管
  - 占地 170 平方米（约 8 个房间）
  - 重量 30 吨
  - 功耗 150 kW
  - 每秒 5,000 次加法运算
- **1948 年**: 晶体管发明（Bell 实验室）
- **1958 年**: 集成电路 (IC) 发明（Jack Kilby）
- **1960s**: 小规模集成 (SSI) —— 每片数十个晶体管
- **1970s**: 中规模集成 (MSI) —— 每片数百个晶体管
- **1980s**: 大规模集成 (LSI) —— 每片数万晶体管
- **1990s 至今**: 超大规模集成 (VLSI) —— 每片数百万至数十亿晶体管

### 数字集成电路分类

#### 按集成度

| 类型 | 缩写 | 集成度 |
|------|------|--------|
| 小规模集成 | SSI | < 10 门/片 |
| 中规模集成 | MSI | 10~100 门/片 |
| 大规模集成 | LSI | 100~10,000 门/片 |
| 超大规模集成 | VLSI | > 10,000 门/片 |

#### 按逻辑器件类型

- **TTL** (Transistor-Transistor Logic) —— 晶体管-晶体管逻辑
  - 双极型晶体管
  - 速度快，功耗较大
  - 标准电源电压：+5V
- **CMOS** (Complementary Metal-Oxide-Semiconductor) —— 互补金属氧化物半导体
  - 场效应管
  - 功耗低，集成度高
  - 目前最主流的数字电路技术
  - 电源电压范围宽（1.8V ~ 5V 等）

| 特性 | TTL | CMOS |
|------|-----|------|
| 基本元件 | 双极型晶体管 (BJT) | 场效应管 (MOSFET) |
| 功耗 | 较高 | 极低（静态接近 0） |
| 集成度 | 较低 | 很高 |
| 抗噪能力 | 中等 | 好 |
| 速度 | 快 | 快（现代工艺） |
| 电源电压 | 5V | 宽范围 |

## 1.2 数字逻辑器件的发展

### 可编程逻辑器件 (PLD)

PLD (Programmable Logic Device) 是一种可由用户编程实现逻辑功能的器件：

- **PROM** (Programmable Read-Only Memory)
- **PLA** (Programmable Logic Array)
- **PAL** (Programmable Array Logic)
- **GAL** (Generic Array Logic)
- **CPLD** (Complex Programmable Logic Device)
- **FPGA** (Field-Programmable Gate Array)

#### PLD 的优势

- 设计周期短
- 可现场编程
- 可反复修改
- 集成度高
- 成本低（小批量）

### 数字系统设计方法

| 传统方法 | 现代方法 |
|----------|----------|
| 分立元件 + 中小规模 IC | PLD + HDL |
| 手工设计、布线 | EDA 工具自动设计 |
| 修改困难 | 可灵活修改 |
| 电路板面积大 | 芯片集成度高 |
| 可靠性一般 | 可靠性高 |

## 1.3 EDA 技术

### EDA 概述

**EDA** (Electronics Design Automation) —— 电子设计自动化

利用计算机软件完成数字系统的设计、仿真和验证，极大提高了设计效率和可靠性。

### 硬件描述语言 (HDL)

**HDL** (Hardware Description Language) 是一种用文本形式描述数字电路结构和行为的语言。

#### 主流 HDL

| 语言 | 特点 |
|------|------|
| **VHDL** | 语法严谨，类似 Ada，1987 年 IEEE 标准 |
| **Verilog HDL** | 语法灵活，类似 C 语言，1995 年 IEEE 标准 |
| **SystemVerilog** | Verilog 的增强版，支持更多抽象层次 |

> 本课程使用 **Verilog HDL** 进行数字电路设计。
