## 2.4 逻辑门电路概述

逻辑门是数字电路的基本构建单元，实现基本的逻辑运算（与、或、非、与非、或非等）。

### 常见逻辑门电路类型

| 类型 | 全称 | 特点 |
|------|------|------|
| DTL | Diode-Transistor Logic | 二极管-晶体管逻辑（早期） |
| **TTL** | Transistor-Transistor Logic | 晶体管-晶体管逻辑（速度快） |
| ECL | Emitter-Coupled Logic | 发射极耦合逻辑（超高速） |
| MOS | Metal-Oxide-Semiconductor Logic | 金属氧化物半导体逻辑 |
| **CMOS** | Complementary MOS | 互补 MOS（功耗低） |

---

## 2.4.1 半导体开关特性

### 理想开关

```text
        S
vI ────┤o──── vO    闭合 (S=1): vO = vI
                      断开 (S=0): vO = 高阻
```

理想开关特性：
- 闭合时电阻为 0，压降为 0
- 断开时电阻无穷大，漏电流为 0
- 开关速度无限快

### 二极管开关

- **正向偏置**：导通，压降约 0.7V（硅管）
- **反向偏置**：截止，只有微小漏电流

### 双极型晶体管 (BJT) 开关

```text
       VCC
        │
       Rc
        │
   RB   iC
    │   │
vI ─┤├─ vCE
    │   │
   iB   │
        │
       ─┴─ GND
```

BJT 的三种工作状态：

| 状态 | 条件 | 特点 |
|------|------|------|
| **截止区** | $v_I < V_{ON}$（约 0.5V） | $i_B=0$, $i_C=0$, $v_O=V_{CC}$ |
| **放大区** | $v_I > V_{ON}$, $i_B < I_{BS}$ | $i_C = \beta i_B$, $v_O$ 随 $v_I$ 变化 |
| **饱和区** | $v_I > V_{ON}$, $i_B > I_{BS}$ | $v_{CE} \approx 0.3V$, $v_O \approx 0V$ |

### MOS 管开关

| 类型 | 符号 | 导通条件 | 特点 |
|------|------|----------|------|
| NMOS | N 沟道 | $V_{GS} > V_{TH(N)}$ | 导通时 $V_{DS} \approx 0$ |
| PMOS | P 沟道 | $V_{GS} < V_{TH(P)}$ | 导通时 $V_{SD} \approx 0$ |

## 2.4.2 TTL 逻辑门

### TTL 与非门电路结构

```text
            +5V(Vcc)
              │
              R1
              │
ABC ─┤├──────┬──── R2 ──┬─── R4 ──┐
    多发射极T1│          │   T3   │
              │    T2    │     │   │
              │    │     │   D1   │
              │    R3     R5   │   │
              │    │     │   │   │
              └────┴─────┴───┴───┴─── Y
                          │   │
                          T4   T5
                          │   │
                          └───┴─── GND
```

#### 工作原理

**输入全高 (3.6V)**：
- T1 基极约 2.1V
- T1 反向放大，T2、T5 饱和导通
- T3 截止，T4 截止
- $v_O = V_{CE(sat)} \approx 0.3V$（输出低电平）

**输入有低 (0.3V)**：
- T1 基极约 1V
- T1 饱和导通，T2、T5 截止
- T3、T4 导通
- $v_O = V_{CC} - i_{B3}R_2 - V_{BE3} - V_{D1} \approx 3.6V$（输出高电平）

**逻辑功能**：$Y = \overline{ABC}$（**三输入与非门**）

### 电压传输特性

```text
 vO(V)
   │
3.6├    UOH
   │     ──────────┐
   │               │
   │               │
   │               └──────────
0.3├                           UOL
   └──┬────┬────────────────── vI(V)
     UIL Uoff    Uon  UIH
     0.3 0.8    1.8   3.6
```

#### 主要电压参数

| 参数 | 符号 | 典型值 | 说明 |
|------|------|--------|------|
| 输出高电平 | $V_{OH}$ | 3.6V | 标准低 > 2.4V |
| 输出低电平 | $V_{OL}$ | 0.3V | 标准高 < 0.4V |
| 输入高电平 | $V_{IH}$ | 3.6V | 标准 > 2.0V |
| 输入低电平 | $V_{IL}$ | 0.3V | 标准 < 0.8V |
| 开门电平 | $V_{ON}$ | 1.8V | 保证输出为低的最小输入电平 |
| 关门电平 | $V_{OFF}$ | 0.8V | 保证输出为高的最大输入电平 |

### 噪声容限 (Noise Margin)

**低电平噪声容限**：
$$V_{NL} = V_{OFF} - V_{IL} = 0.8V - 0.3V = 0.5V$$

**高电平噪声容限**：
$$V_{NH} = V_{IH} - V_{ON} = 3.6V - 1.8V = 1.8V$$

### 扇出系数 (Fan-out)

扇出系数是指一个门能驱动的同类门的最大数目。

**低电平扇出**：
$$N_{OL} = \frac{I_{OL(max)}}{I_{IL(max)}}$$

**高电平扇出**：
$$N_{OH} = \frac{I_{OH(max)}}{I_{IH(max)}}$$

通常 TTL 门的扇出系数为 8~10。

### 传输延迟 (Propagation Delay)

```text
vI  ──────┐   ┌──────
           │   │
           └───┘   t
vO  ──┐   ┌───────
       │   │
       └───┘       t
       ↑   ↑
      tPHL tPLH
```

| 参数 | 符号 | 说明 |
|------|------|------|
| 上升沿延迟 | $t_{PLH}$ | 输入变化到输出从低到高 |
| 下降沿延迟 | $t_{PHL}$ | 输入变化到输出从高到低 |
| 平均延迟 | $t_{pd}$ | $(t_{PHL} + t_{PLH}) / 2$ |

> TTL 典型传输延迟为 5~30ns。

### 功耗 (Power Dissipation)

$$P(avg) = V_{CC} \cdot I_{CC}(avg) = V_{CC} \cdot \frac{I_{CCH} + I_{CCL}}{2}$$

> TTL 门典型功耗约 10mW。

## 2.4.3 集电极开路门 (OC 门)

### OC 门结构

OC (Open Collector) 门去掉了 TTL 图腾柱输出的上拉管 T3、T4，输出管 T5 的集电极开路。

```text
       Vcc
        │
  A ──┤  ├──┬─── Y   外部需要接上拉电阻 RL
  B ──┤  ├──┤   到 V'cc
            │
            T5
            │
           GND
```

### 线与逻辑 (Wired-AND)

多个 OC 门的输出可以直接相连，实现"线与"功能：

$$Y = Y_1 \cdot Y_2 = \overline{AB} \cdot \overline{CD} = \overline{AB + CD}$$

```text
   Vcc      V'cc
    │        │
    │       RL
    │        │
  A ─┤  ├──┼────── Y
  B ─┤  ├──┤
    │
  C ─┤  ├──┼──────
  D ─┤  ├──┤
            │
           GND
```

### 上拉电阻 RL 的选择

- **RL 过小**: 灌电流过大，$V_{OL}$ 可能超过允许值
- **RL 过大**: 拉电流不足，$V_{OH}$ 可能低于允许值

> 通常 RL 在几百欧到几千欧之间选择。

## 2.4.4 三态门 (Tri-state Gate)

### 三态门结构

三态门在普通逻辑门的基础上增加了**使能控制端 (EN)**：

```text
  EN ──┤控制├──
        │
  A ──┤  ├── Y
  B ──┤  ├──
```

### 输出状态

| EN | 功能 | 输出状态 |
|----|------|----------|
| 1 | 正常工作 | 0 或 1 |
| 0 | 禁止 | 高阻态 Z |

或（控制电平极性可不同）：

| EN | 功能 | 输出状态 |
|----|------|----------|
| 0 | 正常工作 | 0 或 1 |
| 1 | 禁止 | 高阻态 Z |

### 三态门的应用

#### 1. 总线结构 (Bus System)

多个三态门的输出可以连接到同一根总线上，通过分时使能实现数据共享：

```text
A1 ──┤  ├──┐
B1 ──┤  ├──┤
 EN1 ────┤ ├──┬─────── DATA BUS
A2 ──┤  ├──┤ ├───┐
B2 ──┤  ├──┤    │   │
 EN2 ────┤ ├────┘   │
 ...               │
Ai ──┤  ├──────────────┘
Bi ──┤  ├──
 ENi ────┤
```

> 在同一时刻，**只能有一个三态门使能**，否则会发生总线冲突。

#### 2. 双向数据传输

```text
          ┌──┬──┬──┐
 DATA ────┤  │  │  ├────
          └──┴──┴──┘
           EN0 EN1 EN2
```

#### 3. 数据选择器/多路复用

### 三态门 vs OC 门

| 特性 | OC 门 | 三态门 |
|------|-------|--------|
| 输出状态 | 0、1 | 0、1、高阻 Z |
| 线与功能 | 支持 | 通常不支持 |
| 总线应用 | 较少 | 广泛 |
| 速度 | 较慢（受 RL 影响） | 较快 |
| 是否需要外接电阻 | 需要上拉电阻 | 不需要 |

## 2.4.5 CMOS 逻辑门

### CMOS 反相器

CMOS (Complementary MOS) 反相器由一对互补的 PMOS 和 NMOS 构成：

```text
          VDD
           │
      ┌────┤
      │   T1 (PMOS)
      │   G1
vI ──┼────┤
      │   S1
      │    │
      │    └────── vO
      │   S2
      │    │
      └────┤
          G2
      │   T2 (NMOS)
      │
     GND
```

#### 工作原理

| $v_I$ | $v_{GS1}$ (PMOS) | $v_{GS2}$ (NMOS) | T1 | T2 | $v_O$ |
|-------|------------------|------------------|----|----|-------|
| 0V (低) | $-V_{DD}$ (导通) | 0V (截止) | ON | OFF | $V_{DD}$ (高) |
| $V_{DD}$ (高) | 0V (截止) | $V_{DD}$ (导通) | OFF | ON | 0V (低) |

**逻辑功能**：$Y = \overline{A}$

#### CMOS 反相器的优势

- **静态功耗极低**：CMOS 管在静态时总有一个截止，几乎没有电流
- **噪声容限大**：约 40% $V_{DD}$
- **电源电压范围宽**：1.8V ~ 5V 甚至更宽
- **扇出大**：输入阻抗极高
- **抗干扰能力强**

### CMOS 与非门

```text
          VDD
           │
  A ──┤├── PMOS1     PMOS 并联
  B ──┤├── PMOS2        │
           │            │
           └────────────┘
                    │
                    └───────── Y
                    │
           ┌────────┘
  A ──┤├── NMOS1     NMOS 串联
  B ──┤├── NMOS2        │
           │
          GND
```

**逻辑功能**：$Y = \overline{AB}$

### CMOS 或非门

```text
          VDD
           │
  A ──┤├── PMOS1     PMOS 串联
  B ──┤├── PMOS2        │
           │
           └───────────── Y
                    │
  A ──┤├── NMOS1     │
           │         │
  B ──┤├── NMOS2   NMOS 并联
           │
          GND
```

**逻辑功能**：$Y = \overline{A + B}$

### CMOS 传输门 (Transmission Gate)

由 PMOS 和 NMOS 并联构成，可传输 0~VDD 全摆幅信号：

```text
           C
            │
    ┌───────┼─────── PMOS ────────┐
IN ─┤                              ├─── OUT
    └─────── NMOS ──────────────┘
            │
           C
```

- 当 $C=1$, $\overline{C}=0$: 传输门导通
- 当 $C=0$, $\overline{C}=1$: 传输门断开（高阻）

## TTL 与 CMOS 对比

| 特性 | TTL | CMOS |
|------|-----|------|
| 基本元件 | 双极型晶体管 (BJT) | 场效应管 (MOSFET) |
| 功耗 | 较高 (~10mW/门) | 极低 (~μW/门，静态) |
| 集成度 | 较低 | 很高 |
| 电源电压 | 5V（固定） | 宽范围 (1.8~5V) |
| 噪声容限 | 中等 (~0.4V) | 好 (~30% VDD) |
| 扇出 | 8~10 | >50 |
| 传输延迟 | 5~30ns | 与 TTL 相当，甚至更快 |
| 抗干扰 | 一般 | 较好 |
| 成本 | 较高 | 较低（现代工艺） |
