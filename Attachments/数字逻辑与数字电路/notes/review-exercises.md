## 一、组合逻辑分析与设计

### 1. 74LS138 译码器实现逻辑函数

用 74LS138 和与非门实现以下函数：

**(1)** `f(A1, B2, C3) = sum(m(0,2,3,4,5,7))`

**(2)** `f(A1, B2, C3) = sum(m(1,2,3,5,6))`

**方法：**
- 将 A,B,C 连接到 74LS138 的地址输入 A2,A1,A0
- 使能端 STA=1, STB=STC=0
- 将对应最小项的输出通过与非门连接到 F

参考电路：

```
         F
         |
         &
  Y0 Y1 Y2 Y3 Y4 Y5 Y6 Y7
  |  |  |  |  |  |  |  |
 └──┴──┴──┴──┴──┴──┴──┘
         74LS138
        A2 A1 A0 STA STB STC
        C3 B2 A1  1
```

### 2. 用 74LS138 实现全加器

全加器的逻辑表达式：
```
Di = sum(m(1,2,4,7))
Ci+1 = sum(m(3,5,6,7))
```

将 Ai, Bi, Ci 连接到 74LS138 的地址输入，分别通过与非门得到 Di 和 Ci+1。

### 3. 数据选择器 (74LS151) 实现逻辑函数

用 74LS151 实现：

**(1)** `f(A1, B2, C3) = sum(m(0,2,3,4,5,7))`

**(2)** `f(A1, B2, C3) = sum(m(1,2,3,5,6))`

### 4. 险象分析与消除

判断函数 `F(A,B,C,D) = ABC + ABC + BD` 是否存在险象并消除。

**分析：**
- 当 A=1, B=1, C 变化时 → F = 1*C + 1*C + ... = C + C → 存在静态险象
- 当 B=1, D=1, A,C 变化时 → 可能存在功能险象

**消除：** 在卡诺图中增加冗余项覆盖相切的卡诺圈。

## 二、触发器与时序逻辑

### 1. 触发器特性方程

| 触发器 | 特性方程 |
|--------|----------|
| RS | Qn+1 = S + RQn (RS=0) |
| D | Qn+1 = D |
| JK | Qn+1 = JQn + KQn |
| T | Qn+1 = T XOR Qn |

### 2. 时序电路分析

**分析步骤：**
1. 写出各触发器激励方程
2. 写出状态方程
3. 列出状态表
4. 画出状态图
5. 画出时序图
6. 说明逻辑功能

**示例：** 两个 JK 触发器构成的时序电路

```
J1 = Q0, K1 = 1
J0 = Q1, K0 = 1

Q1n+1 = J1*Q1n + K1*Q1n = Q0n*Q1n
Q0n+1 = J0*Q0n + K0*Q0n = Q1n*Q0n
```

状态转移：

| Q1 | Q0 | Q1n+1 | Q0n+1 |
|----|----|-------|-------|
| 0  | 0  | 0     | 1     |
| 0  | 1  | 1     | 0     |
| 1  | 0  | 0     | 0     |
| 1  | 1  | 0     | 0     |

状态图：`00 → 01 → 10 → 00`（模3计数器，状态11进入00）

### 3. 时序电路设计

**设计步骤：**
1. 建立状态图/状态表
2. 状态化简

4. 选择触发器（D/JK/T）
5. 推导激励方程

#### 示例：检测 "110" 序列

**状态定义：**
- S0: 初始
- S1: 收到 "1"
- S2: 收到 "11"
- S3: 收到 "110"，输出 Z=1

**状态图：**
```
        0/0                    0/0
S0 ──1/0──→ S1 ──1/0──→ S2 ──0/1──→ S3
↑                                       │
└────────────────0/0────────────────────┘
```

**状态表：**

| Q1n Q0n | X=0 | X=1 | Z |
|---------|-----|-----|---|
| 00      | 00  | 01  | 0 |
| 01      | 01  | 10  | 0 |
| 10      | 11  | 01  | 0 |
| 11      | 00  | 01  | 1 |

**激励方程 (D触发器)：**
```
D1 = X(Q1 + Q0)
D0 = Q1Q0 + XQ1
Z = XQ1Q0
```

## 三、Verilog HDL 重点

### 常用 Verilog 语句

```verilog
// 模块结构
module <name>(<ports>);
    // 端口声明
    input [3:0] A;
    output reg [7:0] Y;

    // 连续赋值
    assign S = A ^ B;

    // always 块（组合逻辑）
    always @(A or B)
        Y = A & B;

    // always 块（时序逻辑）
    always @(posedge clk or negedge rst_n)
        if (!rst_n) Q <= 0;
        else Q <= D;

    // 条件语句
    if (条件) ...
    else if (条件) ...
    else ...

    // 多路分支
    case (表达式)
        值1: 语句1;
        值2: 语句2;
        default: 默认语句;
    endcase

    // 循环
    for (i=0; i<N; i=i+1) ...
endmodule
```

### 阻塞赋值与非阻塞赋值

| 赋值 | 符号 | 应用场景 | 执行方式 |
|------|------|----------|----------|
| 阻塞 | `=` | 组合逻辑 | 顺序执行 |
| 非阻塞 | `<=` | 时序逻辑 | 并行执行 |

### 状态机模板

```verilog
always @(posedge clk or posedge rst) begin
    if (rst)
        state <= S0;
    else
        case (state)
            S0: if (X) state <= S1; else state <= S0;
            S1: if (X) state <= S2; else state <= S1;
            ...
            default: state <= S0;
        endcase
end

always @(*) begin
    case (state)
        S0: Z = 0;
        ...
        default: Z = 0;
    endcase
end
```
